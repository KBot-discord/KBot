# Report

The [report command](/commands#report) allows server staff to send a copy of a user's message to a designated channel.

::: tip Required channel permissions
KBot needs these permissions in the set channel to be able to send messages: `View Channel`, `Send Messages`, `Embed Links`.
:::

## Setup

To set a report channel, user [/moderation set](/commands#moderation-set).

## Reporting a message

1. Right-click the message you wish to report
2. Hover over `Apps` in the menu
3. Select `Report`
